### Python Version

2.7.13

### Packages

```bash
pip install sqlalchemy sqlalchemy_utils ccxt
```

BIT_DIGGER_SQL_HOSTNAME

BIT_DIGGER_SQL_PORT

BIT_DIGGER_SQL_DB_NAME
